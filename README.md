# Blinkit Sales Data Analysis | Excel Dashboard

This project analyzes Blinkit's grocery sales dataset to understand product performance, outlet trends, and sales patterns.

Dataset size: 100,000+ rows.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization
- Dashboard Creation

## Key Metrics
- Total Sales
- Average Sales
- Number of Items
- Average Rating

## Key Insights
- Tier 3 outlets generated strong sales performance
- Regular fat items contributed a major share of sales
- Supermarket Type 1 outlets had the highest sales volume
- Compared performance across outlet types and locations
- Analyzed impact of item attributes on sales
